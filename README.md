# 🤖 AI Resume Analyzer

An AI-powered Resume Analyzer built using Flask, MySQL, and Google Gemini AI.

## 📌 Features

- 🔐 User Registration & Login
- 📄 Resume Upload (PDF/DOCX)
- 📊 ATS Score Calculation
- 🎯 Job Description Matching
- 🧠 AI Resume Suggestions (Google Gemini)
- 🏷️ Skill Extraction
- 📈 Interactive Charts
- 📥 PDF Report Generation
- 📱 Responsive User Interface

## 🛠️ Tech Stack

- Python
- Flask
- MySQL
- SQLAlchemy
- HTML
- CSS
- Bootstrap 5
- Chart.js
- Google Gemini API
